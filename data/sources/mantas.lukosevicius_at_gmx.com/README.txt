Usage:

There is a standard-interface physionet2013.m function provided, compatible with standard scoring scripts.

Alternatively:

* b00-99 .dat and .head files should be located in the folder 'duomenys'.

* Running generate_all_B_results.m or generate_all_B_results_std.m generates the annotations b00-99 .fqrs_entry1 in a folder 'annotations'. The annotations are generated in a text format, numbers representing milliseconds.
